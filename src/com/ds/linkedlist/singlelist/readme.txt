Single Linked List -> Linked list is data structure used to store collection of data.

Single Linked List Properties :-
----------------------------------
----------------------------------

1. Data is stored as node (Data, Pointer to Next Node).
2. All the elements are connected by pointer.
3. Last node points to the NULL.
4. Can grow or shrink as required on run time.
5. Does not allocate initial memory as Array. Hence, does not waste memory.
